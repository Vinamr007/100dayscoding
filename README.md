### Prerequisites
- Java Development Kit (JDK) 8 or higher
- A Java IDE (IntelliJ IDEA, Eclipse, VS Code) or command line

### Running the Programs

1. Clone this repository:
```bash
git clone https://github.com/Vinamr007/100dayscoding.git
```

2. Compile a Java file:

Or use your preferred IDE to open and run the Java files directly.
